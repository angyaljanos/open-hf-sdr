Open HF SDR
===========

# Specs
* 0.5-30 MHz input freq range

# Receiver
## First stage
* 30 MHz low-pass filter
* Programmable auttenator with 10, 20, and 30 dB options
* Highly linear mixer to 37 MHz IF

## Second stage
* IF filter for 37 MHz
* IF amplifier

## Third stage
* Mixer to IQ signals
* IF filter
* IF amplifier
* ADC matching
